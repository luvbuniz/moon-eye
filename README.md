# 🌙 Moon Eye

**A sparkly, girly building world** — pink blocks, arctic snow, flowers everywhere, custom characters, and tons of animals.

**Official repo:** [github.com/luvbuniz/moon-eye](https://github.com/luvbuniz/moon-eye)  
**Play live:** [luvbuniz.github.io/moon-eye](https://luvbuniz.github.io/moon-eye/)

---

## 📱 Tablet: offline + local save

1. Open the live link **once online** (hotspot is fine) so the tablet downloads everything  
2. **iPad:** Share → **Add to Home Screen**  
3. **Android:** Chrome → **Add to Home screen** / Install  
4. Play from the **Moon Eye** icon anytime — **works offline** after that  

**Saves on each tablet separately** (localStorage): your character, hotbar, position, day/night, and whole built world.  
Kids do not share saves across tablets.

## Features

| | |
|--|--|
| **Character** | Make a girl / boy / sparkle friend — name, hair & outfit colors |

## Controls

| Input | Action |
|-------|--------|
| WASD | Move |
| Mouse / look pad | Look |
| Space / C | Fly up / down (fly mode) |
| F | Toggle fly |
| Left click | Place |
| Right click | Break |
| Tab / Search | Everything catalog |
| E | Eat food |
