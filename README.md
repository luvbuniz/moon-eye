# 🌙 Moon Eye

**A sparkly, girly building world** — pink blocks, cherry blossoms, bamboo forests, cute animals, yummy food, and houses in the sky.

Made for dreamers who love animals, pink, and building (Minecraft-style, but cuter).

## What is Moon Eye?

Moon Eye is a browser sandbox where you can:

- **Build** with tons of blocks (lots of pink!)
- **Stack blocks into the sky** and put a house on top
- **Plant** cherry blossom trees and bamboo forests
- **Place water & grass** to grow a pretty world
- **Fence in animals** — foxes, snakes, horses, birds, pandas, cats
- **Eat food** — cake, cookies, strawberries, cupcakes, and more
- **Search** everything in the game with one button
- Put down **chairs, tables, beds, fairy lamps**, fences, glass, clouds, hearts & stars

---

## Play

Open `index.html` in Chrome or Edge, or:

```bash
npx --yes serve .
```

Click **Enter Moon Eye**, then click the game so you can look around.

---

## Controls

| Key / mouse | Action |
|-------------|--------|
| **W A S D** | Move |
| **Mouse** | Look (click game first) |
| **Space** | Jump |
| **F** | Fly mode (creative sky building!) |
| **C** or **Ctrl** | Fly down |
| **Left click** | Place block |
| **Right click** / **Q** | Break block |
| **1–9** | Hotbar slots |
| **E** | Eat selected food |
| **Tab** or **Search** button | Show *everything* |
| **Esc** | Unlock mouse / close search |

---

## Search

Press **Search** (or **Tab**) to open the full catalog:

- Pink & pastel blocks  
- Nature (grass, water, cherry blossom, bamboo…)  
- Furniture (chair, table, bed, lamp)  
- Food  
- Animals (click to spawn a pet nearby!)  

Type in the search box to filter.

---

## Animals

🦊 Fox · 🐍 Snake · 🐴 Horse · 🐦 Bird · 🐼 Panda · 🐱 Cat  

They wander around. Use fences to make a cute pen. Spawn more from Search or the **Spawn Pet** button.

---

## Tips for sky houses

1. Press **F** to fly  
2. Stack **cloud blocks** or pink blocks upward  
3. Build a floor in the sky  
4. Add walls, a chair, a table, cherry blossoms  
5. Break the blocks underneath if you want a floating island
