## About

**Moon Eye** is a free browser building game. You place blocks, decorate with furniture, spawn cute animals, plant cherry blossoms and bamboo, eat snacks, and build floating sky houses.

It was designed from a kid’s wish list: pink everything, animals, a Search button that shows *everything*, fences, food, and stacking blocks all the way to the clouds.

| Feature | Details |
|--------|---------|
| **Blocks** | Pink dream, hot pink, rose quartz, lavender, cotton candy, pearl, mint, sky blue, hearts, stars, clouds… |
| **Beds** | Many colors — pink, hot pink, purple, lavender, sky blue, mint, sunshine, peach, pearl, rainbow, heart |
| **Lights** | Pink/white/lavender/gold/rose candles, fairy lamp, moon lamp, heart lamp, lantern, string lights, glow block, night light |
| **Nature** | Grass, water, cherry blossom trees, bamboo forest, flowers |
| **Furniture** | Chairs, tables, beds, fences |
| **Animals** | Fox, snake, horse, bird, panda, cat |
| **Food** | Cake, cookie, strawberry, cupcake, candy, sandwich, berry juice, ice cream |
| **Day / Night** | Toggle button — night has stars & moon; candles glow beautifully |
| **Search** | One button lists every block, light, food, and animal |

## Controls

| Input | Action |
|-------|--------|
| **W A S D** | Move |
| **Mouse** | Look (click the game first) |
| **Space** | Jump |
| **F** | Fly mode (great for sky houses) |
| **C** / **Ctrl** | Fly down |
| **Left click** | Place block |
| **Right click** / **Q** | Break block |
| **1–9** | Hotbar |
| **E** | Eat selected food |
| **Tab** or **Search** | Open full catalog |
| **Day/Night** button | Toggle sun ☀️ and moon 🌙 |
| **Esc** | Unlock mouse / close search |

## Colorful beds

Open **Search → Furniture** (or search “bed”) and pick:

- Pink · Hot Pink · Purple · Lavender  
- Sky Blue · Mint · Sunshine · Peach · Pearl  
- Rainbow Bed · Heart Bed  

Place them in your house — each has its own blanket & pillow colors!

---

## Candles & lights

Open **Search → Candles & Lights**:

- Candles (pink, white, lavender, gold, rose) with real glow  
- Fairy lamp · Moon lamp · Heart lamp  
- Cute lantern · String lights · Night light · Glow block  

Tip: switch to **night** and light up your house 🕯️

---

## Sky houses

1. Press **F** to fly  
2. Stack **cloud** or pink blocks upward  
3. Build a floor, walls, a bed, and candles  
4. Break the blocks under the floor if you want a floating island
